# 2023年H题
[toc]
## 思路
1. 加法器的搭建 A+B=C
   * 如何保证频率响应特性，使得1-100K信号相加放大倍数不变（电容的使用）
   * C相信号示波器打出来有失真（漂移），如何解决    （不影响采样）
2. 抬升电路（ADC驱动电路）：适配ADC采样
3. ADC采样
   * 采样率的设定 ：fs 512000Hz（供参考）
4. FFT处理，反推频率
5. DAC or DDS 产生波形
   * DDS：保证能调频率，幅度相位要搭配最后一步操作实现同频显示和Vpp大于1V
6. 最后一步，**同频显示**的实现
   * 锁相装置（锁相放大器）
     * [锁相放大器](https://zhuanlan.zhihu.com/p/644133931)的实现也需要AGC，其输入信号应该与参考信号等幅，所以**先放大，再锁相**
   * AGC（压控增益）

## 还没有解决的点
* 加法器频率响应不对，无法保证全频率A+B=C
* C相信号失真
* 锁相放大器
